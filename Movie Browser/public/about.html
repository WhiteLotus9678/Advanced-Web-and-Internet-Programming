About React/Express Template

This project is a template for managing a single application that includes an express.js server with Data API endpoints along with a React.js front-end client application. The files and directories are structured as follows:

  • api - Code for the express.js data API endpoints.
  • client - JSX, CSS, & HTML code for the React.js frontend.
  • dist - Code generated by webpack for the compiled client.
  • public - Statically shared files (media, js, css, etc.).
  • src - Code for the express.js server entry point.
